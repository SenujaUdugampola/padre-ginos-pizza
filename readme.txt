# 🍕 Padre Gino’s Pizza - Online Ordering System

Padre Gino’s Pizza is a simple web-based pizza ordering system that allows customers to browse a menu, customize their orders, manage their cart, and place orders online. This system is designed for small to medium-sized pizzerias looking to digitize their ordering process and improve customer convenience.

---

## 🚀 Features

- View available pizza menu with prices
- Customize pizzas with toppings and sizes
- Add and remove items from the cart
- Real-time price calculation
- Submit and process orders
- Admin dashboard for order management

---

## 🛠️ Technologies Used

- **Frontend:** HTML, CSS, JavaScript, React
- **Backend:** PHP
- **Database:** MySQL
